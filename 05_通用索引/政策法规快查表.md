## 九、常见政策问题解答

### Q1: 资金使用"三公经费"限制是什么？
**A**: 根据相关规定，禁止使用财政资金用于：
- 公务接待、公务用车、公务出国等"三公"支出超出规定
- 详见：《党风廉政建设和反腐败工作的意见》

### Q2: 项目中如何应用政府采购制度？
**A**: 按照以下程序进行采购：
1. 确定采购需求 → 2. 确定采购方式 → 3. 发布采购信息 → 4. 组织评审 → 5. 确定中标人 → 6. 签订合同
- 详见：《政府采购法》及其实施条例

### Q3: 涉农资金补贴如何申报？
**A**: 一般程序为：
1. 符合资格条件 → 2. 提交申报材料 → 3. 部门审查 → 4. 公示 → 5. 确认补贴对象 → 6. 拨付资金
- 具体要求请查看相应的补贴办法

### Q4: 项目不达绩效目标怎么处理？
**A**: 按照以下规定处理：
1. 确认未达成原因 → 2. 进行效果评价 → 3. 提出改进措施 → 4. 可能影响后续项目评价与资金安排
- 详见：《财政支出绩效管理办法》
