# 脱敏规则与使用说明

## 一、脱敏原则
本项目已进行**全面脱敏处理**，遵循“保留审计逻辑、隐去敏感信息”的原则，将具体地域、文号、金额、单位名称等转化为通用审计术语，确保内容可公开传播且符合保密要求。

## 二、核心替换规则
| 原始类型 | 脱敏后表述 | 说明 |
|:---|:---|:---|
| 省级地域/部门 | 省级 / 省级审计机关 / 省级农业农村主管部门 | 隐去具体省份名称 |
| 市级地域/部门 | 市级 / 市级审计机关 / 市级财政部门 | 隐去具体地市名称 |
| 区县级 | 区县级 / 乡镇 / 村级 | 隐去具体区县名称 |
| 政策文号 | 财农〔202X〕XX号 / 农规〔202X〕XX号 | 隐去具体年份与序号 |
| 法规名称 | 《省级乡村振兴促进条例》 / 《市级专项资金管理办法》 | 保留法规性质，隐去地域 |
| 编制单位 | 审计机关（参考模板） | 统一表述 |
| 时间/金额 | 202X年X月X日 / 达到法定限额 / 大额资金 | 保留审计阈值逻辑，隐去具体数值 |

## 三、使用建议
1. **本地化适配**：实际审计项目中，请将“省级/市级”替换为实际层级，并补充当地最新政策文号。
2. **阈值校准**：文中“达到法定限额”“超过合同价10%”等为通用审计标准，具体执行请参照当地现行规定。
3. **保密要求**：严禁在公开仓库中上传含真实项目名称、被审计单位、涉密金额或人员信息的底稿。
