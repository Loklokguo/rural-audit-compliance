# rural-audit-compliance

🌾 **乡村振兴审计合规助手** - 基于实务经验的审计方法论沉淀

## 📖 项目介绍

本项目构建"**政策-资金-项目-绩效**"四位一体的审计模型，聚焦资金安全、政策落实与利益输送风险防控，助力提升涉农资金审计质效。

**核心目标**：
- 📋 提供标准化的审计实施方案
- ✅ 建立可复用的合规审查清单
- 🔍 识别常见风险点与典型案例
- 📜 汇总相关政策法规依据

## 🎯 主要功能

### 1. 审计方案编制
根据不同项目类型，快速生成结构化的审计实施方案，包含：
- 审计目标与范围
- 审计重点与风险识别
- 审计步骤与时间安排
- 资源配置建议

### 2. 合规审查辅助
提供分阶段的标准化审查清单：
- **资金环节**：预算编制、拨付、使用、结算
- **项目环节**：立项、采购、实施、验收
- **运营环节**：内部控制、廉政风险、绩效管理

### 3. 风险识别与防控
识别常见风险点：
- 💰 资金挪用、截留、套取
- 🔄 利益输送、权力寻租
- 📝 程序违规、档案不完整
- 📊 绩效虚假、数据造假

### 4. 政策依据匹配
提供通用型政策法规适用对照表：
- 国家审计准则
- 农业相关管理办法
- 财政资金使用制度
- 廉政建设要求

## 🚀 快速开始

### 场景一：我需要编制一份审计方案
👉 查看 `01_审计实施方案/` 目录，选择对应项目类型的模板进行填充。

### 场景二：我需要对项目进行合规审查
👉 查看 `04_审查要点清单/` 目录，根据项目阶段选择相应清单。

### 场景三：我需要识别风险点
👉 查看 `03_审计指南/` 中的"风险识别指南"或 `05_通用索引/` 中的"常见风险点索引"。

### 场景四：我需要找政策依据
👉 查看 `02_政策法规依据/` 中的相应文档或 `05_通用索引/` 中的"政策法规快查表"。

## 💡 核心特点

✨ **规范化**：结构清晰，易于复用  
🔒 **脱敏化**：所有内容已进行脱敏处理，无具体地域/单位/金额信息  
💼 **实务化**：内容贴合一线审计实务，具备可操作性  
🛡️ **合规化**：严格遵循国家审计准则和廉政建设要求
